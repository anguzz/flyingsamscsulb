# Flying Samaritans CSULB Website (Legacy)

This repository contains the previous Svelte-based website for the Flying Samaritans CSULB chapter.

## Status

This site is no longer actively maintained. 

The club website has been migrated to Google Sites for easier long-term management by non-technical officers.

## Official Website

 [https://flyingsamscsulb.com](https://flyingsamscsulb.com)

## Reason for Migration

The previous site was built using:

* Svelte
* GitHub
* Vercel

While functional, it required technical knowledge for updates, deployments, and domain management. To ensure continuity and ease of maintenance for future club leadership, the site was transitioned to Google Sites.

## Archive Purpose

This repository is retained for:

* Historical reference
* Design/content backup
* Potential future redevelopment
